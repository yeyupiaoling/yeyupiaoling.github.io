《PaddlePaddle从入门到炼丹》四——卷积神经网络

前言

上一章我们通过学习线性回归例子入门了深度学习，同时也熟悉了PaddlePaddle的使用方式，那么我们在本章学习更有趣的知识点卷积神经网络。深度学习之所以那么流行，很大程度上是得益于它在计算机视觉上得到非常好的效果，而在深度学习上几乎是使用卷积神经网络来提取图像的特征的。在PaddlePaddle上如何定义一个卷积神经网络，并使用它来完成一个图像识别的任务呢。在本章我们通过学习MNIST图像数据集的分类例子，来掌握卷积神经网络的使用。

训练模型

创建一个mnist_classification.py文件，首先导入所需得包，这次使用到了MNIST数据集接口，也使用了处理图像得工具包。

1import numpy as np
2import paddle as paddle
3import paddle.dataset.mnist as mnist
4import paddle.fluid as fluid
5from PIL import Image
6import matplotlib.pyplot as plt

在图像识别上，使用得算法也经过了多次的迭代更新，比如多层感知器，在卷积神经网络广泛使用之前，多层感知器在图像识别上是非常流行的，从这方面来看，多层感知器在当时也是有一定的优势的。那么如下使用PaddlePaddle来定义一个多层感知器呢，我们可以来学习一下。以下的代码判断就是定义一个简单的多层感知器，一共有三层，两个大小为100的隐层和一个大小为10的输出层，因为MNIST数据集是手写0到9的灰度图像，类别有10个，所以最后的输出大小是10。最后输出层的激活函数是Softmax，所以最后的输出层相当于一个分类器。加上一个输入层的话，多层感知器的结构是：输入层-->>隐层-->>隐层-->>输出层。

1# 定义多层感知器
2def multilayer_perceptron(input):
3    # 第一个全连接层，激活函数为ReLU
4    hidden1 = fluid.layers.fc(input=input, size=100, act='relu')
5    # 第二个全连接层，激活函数为ReLU
6    hidden2 = fluid.layers.fc(input=hidden1, size=100, act='relu')
7    # 以softmax为激活函数的全连接输出层，大小为label大小
8    fc = fluid.layers.fc(input=hidden2, size=10, act='softmax')
9    return fc

卷积神经网络普遍用在图像特征提取上，一些图像分类、目标检测、文字识别几乎都回使用到卷积神经网络作为图像的特征提取方式。卷积神经网络通常由卷积层、池化层和全连接层，有时还有Batch Normalization层和Dropout层。下面我们就创建一个简单卷积神经网络，一共定义了5层，加上输入层的话，它的结构是：输入层-->>卷积层-->>池化层-->>卷积层-->>池化层-->>输出层。我们可以通过调用PaddlePaddle的接口fluid.layers.conv2d()来做一次卷积操作，我们可以通过num_filters参数设置卷积核的数量，通过filter_size设置卷积核的大小，还有通过stride来设置卷积操作时移动的步长。使用fluid.layers.pool2d()接口做一次池化操作，通过参数pool_size可以设置池化的大小，通过参数pool_stride设置池化滑动的步长，通过参数pool_type设置池化的类型，目前有最大池化和平均池化，下面使用的时最大池化，当值为avg时是平均池化。

 1# 卷积神经网络
 2def convolutional_neural_network(input):
 3    # 第一个卷积层，卷积核大小为3*3，一共有32个卷积核
 4    conv1 = fluid.layers.conv2d(input=input,
 5                                num_filters=32,
 6                                filter_size=3,
 7                                stride=1)
 8
 9    # 第一个池化层，池化大小为2*2，步长为1，最大池化
10    pool1 = fluid.layers.pool2d(input=conv1,
11                                pool_size=2,
12                                pool_stride=1,
13                                pool_type='max')
14
15    # 第二个卷积层，卷积核大小为3*3，一共有64个卷积核
16    conv2 = fluid.layers.conv2d(input=pool1,
17                                num_filters=64,
18                                filter_size=3,
19                                stride=1)
20
21    # 第二个池化层，池化大小为2*2，步长为1，最大池化
22    pool2 = fluid.layers.pool2d(input=conv2,
23                                pool_size=2,
24                                pool_stride=1,
25                                pool_type='max')
26
27    # 以softmax为激活函数的全连接输出层，大小为label大小
28    fc = fluid.layers.fc(input=pool2, size=10, act='softmax')
29    return fc

定义输入层，输入的是图像数据。图像是28*28的灰度图，所以输入的形状是[1, 28, 28]，如果图像是32*32的彩色图，那么输入的形状是[3. 32, 32]，因为灰度图只有一个通道，而彩色图有RGB三个通道。理论上它还有一个维度是Batch的，不过这个是PaddlePaddle帮我们默认设置的，我们可以不用理会。

1# 定义输入层
2image = fluid.layers.data(name='image', shape=[1, 28, 28], dtype='float32')
3label = fluid.layers.data(name='label', shape=[1], dtype='int64')

上面定义了多层感机器和卷积神经网络，我们可以在这里调用定义好的网络来获取分类器，读者可以尝试这两种不同的网络进行训练，观察一下他们的准确率如何。

1# 获取分类器
2# model = multilayer_perceptron(image)
3model = convolutional_neural_network(image)

接着是定义损失函数，这次使用的是交叉熵损失函数，该函数在分类任务上比较常用。定义了一个损失函数之后，还有对它求平均值，因为定义的是一个Batch的损失值。同时我们还可以定义一个准确率函数，这个可以在我们训练的时候输出分类的准确率。

1# 获取损失函数和准确率函数
2cost = fluid.layers.cross_entropy(input=model, label=label)
3avg_cost = fluid.layers.mean(cost)
4acc = fluid.layers.accuracy(input=model, label=label)

然后我们从主程序中克隆一个程序作为预测程序，之后可以使用这个预测程序预测测试的准确率和预测自己的图像。

1# 获取测试程序
2test_program = fluid.default_main_program().clone(for_test=True)

接着是定义优化方法，这次我们使用的是Adam优化方法，同时指定学习率为0.001。

1# 定义优化方法
2optimizer = fluid.optimizer.AdamOptimizer(learning_rate=0.001)
3opts = optimizer.minimize(avg_cost)

定义读取MNIST数据集的reader，指定一个Batch的大小为128，也就是一次训练128张图像。

1# 获取MNIST数据
2train_reader = paddle.batch(mnist.train(), batch_size=128)
3test_reader = paddle.batch(mnist.test(), batch_size=128)

接着也是定义一个执行器和初始化参数，Fluid版本使用的流程都差不多。

1# 定义一个使用CPU的执行器
2place = fluid.CPUPlace()
3exe = fluid.Executor(place)
4# 进行参数初始化
5exe.run(fluid.default_startup_program())

输入的数据维度是图像数据和图像对应的标签，每个类别的图像都要对应一个标签，这个标签是从0递增的整型数值。

1# 定义输入数据维度
2feeder = fluid.DataFeeder(place=place, feed_list=[image, label])

最后就可以开始训练了，我们这次训练5个Pass，读者可以根据自己的情况自由设置。在上面我们已经定义了一个求准确率的函数，所以我们在训练的时候让它输出当前的准确率，计算准确率的原理很简单，就是把训练是预测的结果和真实的值比较，求出准确率。每一个Pass训练结束之后，再进行一次测试，使用测试集进行测试，并求出当前的Cost和准确率的平均值。

 1# 开始训练和测试
 2for pass_id in range(5):
 3    # 进行训练
 4    for batch_id, data in enumerate(train_reader()):
 5        train_cost, train_acc = exe.run(program=fluid.default_main_program(),
 6                                        feed=feeder.feed(data),
 7                                        fetch_list=[avg_cost, acc])
 8        # 每100个batch打印一次信息
 9        if batch_id % 100 == 0:
10            print('Pass:%d, Batch:%d, Cost:%0.5f, Accuracy:%0.5f' %
11                  (pass_id, batch_id, train_cost[0], train_acc[0]))
12
13    # 进行测试
14    test_accs = []
15    test_costs = []
16    for batch_id, data in enumerate(test_reader()):
17        test_cost, test_acc = exe.run(program=test_program,
18                                      feed=feeder.feed(data),
19                                      fetch_list=[avg_cost, acc])
20        test_accs.append(test_acc[0])
21        test_costs.append(test_cost[0])
22    # 求测试结果的平均值
23    test_cost = (sum(test_costs) / len(test_costs))
24    test_acc = (sum(test_accs) / len(test_accs))
25    print('Test:%d, Cost:%0.5f, Accuracy:%0.5f' % (pass_id, test_cost, test_acc))

预测图像

训练完成之后，我们可以使用从主程序中克隆的test_program来预测我们自己的图像。再预测之前，要对图像进行预处理，处理方式要跟训练的时候一样。首先进行灰度化，然后压缩图像大小为28*28，接着将图像转换成一维向量，最后再对一维向量进行归一化处理。

1# 对图片进行预处理
2def load_image(file):
3    im = Image.open(file).convert('L')
4    im = im.resize((28, 28), Image.ANTIALIAS)
5    im = np.array(im).reshape(1, 1, 28, 28).astype(np.float32)
6    im = im / 255.0 * 2.0 - 1.0
7    return im
