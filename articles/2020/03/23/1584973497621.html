在Android手机上使用MACE实现图像分类

2020-03-23
   
Android 人工智能 夜雨飘零 深度学习
    0 0

前言

在之前笔者有介绍过《在Android设备上使用PaddleMobile实现图像分类》，使用的框架是百度开源的PaddleMobile。在本章中，笔者将会介绍使用小米的开源手机深度学习框架MACE来实现在Android手机实现图像分类。

MACE的GitHub地址：https://github.com/XiaoMi/mace

编译MACE库和模型

编译MACE库和模型有两种方式，一种是在Ubuntu本地上编译，另一种是使用docker编译。下面就介绍使用这两种编译方式。

使用Ubuntu编译

使用Ubuntu编译源码比较麻烦的是就要自己配置环境，所以下面我们就来配置一下环境。以下是官方给出的环境依赖：

  • 编译完成之后，可以在mace/mace/examples/android/macelibrary/src/main/cpp/看到多了3个文件：

  3. model是存放模型的文件夹，比如我们编译的MobileNet V2模型

使用Docker编译

  • 首先安装docker，命令如下：
1apt-get install docker.io
  • 拉取mace镜像：
1docker pull registry.cn-hangzhou.aliyuncs.com/xiaomimace/mace-dev
  • 获取MACE源码，并按照上一步修改mace/mace/examples/android/目录下的build.sh和mobilenet.yml这个两个文件。
1git clone https://github.com/XiaoMi/mace.git
  • 进入到MACE的根目录，执行以下命令：
1docker run -it -v $PWD:/mace registry.cn-hangzhou.aliyuncs.com/xiaomimace/mace-dev
  • 接着执行以下命令：
1cd mace/mace/examples/android/
2./build.sh

执行之后便可得到跟上一步获取的一样的文件。使用docker就简单很多，少了很多安装依赖环境的步骤。

 73    // Compress the image to the size of the training image
 74    public static float[] getScaledMatrix(Bitmap bitmap, int desWidth,
 75                                          int desHeight) {
 76        // create data buffer
 77        float[] floatValues = new float[desWidth * desHeight * 3];
 78        FloatBuffer floatBuffer = FloatBuffer.wrap(floatValues, 0, desWidth * desHeight * 3);
 79        floatBuffer.rewind();
 80        // get image pixel
 81        int[] pixels = new int[desWidth * desHeight];
 82        Bitmap bm = Bitmap.createScaledBitmap(bitmap, desWidth, desHeight, false);
 83        bm.getPixels(pixels, 0, bm  .getWidth(), 0, 0, desWidth, desHeight);
 84        // pixel to data
 85        for (int clr : pixels) {
 86            floatBuffer.put((((clr >> 16) & 0xFF) - 128f) / 128f);
 87            floatBuffer.put((((clr >> 8) & 0xFF) - 128f) / 128f);
 88            floatBuffer.put(((clr & 0xFF) - 128f) / 128f);
 89        }
 90        if (bm.isRecycled()) {
 91            bm.recycle();
 92        }
 93        return floatBuffer.array();
 94    }

 96    // compress picture
 97    public static Bitmap getScaleBitmap(String filePath) {
 98        BitmapFactory.Options opt = new BitmapFactory.Options();
 99        opt.inJustDecodeBounds = true;
100        BitmapFactory.decodeFile(filePath, opt);
101
102        int bmpWidth = opt.outWidth;
103        int bmpHeight = opt.outHeight;
104
105        int maxSize = 500;
106
107        // compress picture with inSampleSize
108        opt.inSampleSize = 1;
109        while (true) {
110            if (bmpWidth / opt.inSampleSize < maxSize || bmpHeight / opt.inSampleSize < maxSize) {
111                break;
112            }
113            opt.inSampleSize *= 2;
114        }
115        opt.inJustDecodeBounds = false;
116        return BitmapFactory.decodeFile(filePath, opt);
117    }

172    //  predict image
173    private void predict_image(String image_path) {
174        // picture to float array
175        Bitmap bmp = PhotoUtil.getScaleBitmap(image_path);
176        float[] inputData = PhotoUtil.getScaledMatrix(bmp, ddims[2], ddims[3]);
177        try {
178            // Data format conversion takes too long
179            // Log.d("inputData", Arrays.toString(inputData));
180            long start = System.currentTimeMillis();
181            // get predict result
182            float[] result = JniMaceUtils.maceMobilenetClassify(inputData);
183            long end = System.currentTimeMillis();
184            Log.d(TAG, "origin predict result:" + Arrays.toString(result));
185            long time = end - start;
186            Log.d("result length", String.valueOf(result.length));
187            // show predict result and time
188            int r = get_max_result(result);
189            String show_text = "result：" + r + "\nname：" + resultLabel.get(r) + "\nprobability：" + result[r] + "\ntime：" + time + "ms";
190            result_text.setText(show_text);
191        } catch (Exception e) {
192            e.printStackTrace();
193        }

196    // get max probability label
197    private int get_max_result(float[] result) {
198        float probability = result[0];
199        int r = 0;
200        for (int i = 0; i < result.length; i++) {
201            if (probability < result[i]) {
202                probability = result[i];
203                r = i;
204            }
205        }
206        return r;
207    }

211
212        List<String> permissionList = new ArrayList<>();

222            permissionList.add(Manifest.permission.READ_EXTERNAL_STORAGE);
223        }
224
225        // if list is not empty will request permissions
226        if (!permissionList.isEmpty()) {
227            ActivityCompat.requestPermissions(this, permissionList.toArray(new String[permissionList.size()]), 1);
