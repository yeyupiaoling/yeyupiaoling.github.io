前言

VisualDL是一个面向深度学习任务设计的可视化工具，包含了scalar、参数分布、模型结构、图像可视化等功能。可以这样说：“所见即所得”。我们可以借助VisualDL来观察我们训练的情况，方便我们对训练的模型进行分析，改善模型的收敛情况。

  1. scalar，趋势图，可用于训练测试误差的展示
  2. image, 图片的可视化，可用于卷积层或者其他参数的图形化展示
  3. histogram, 用于参数分布及变化趋势的展示
  4. graph，用于训练模型结构的可视化

    以上的图像来自VisualDL的Github

既然那么方便，那么我们就来尝试一下吧。VisualDL底层采用C++编写，但是它在提供C++ SDK的同时，也支持Python SDK，我们主要是使用Python的SDK。顺便说一下，VisualDL除了支持PaddlePaddle,之外，还支持pytorch, mxnet在内的大部分主流DNN平台。

VisualDL的安装

本章只讲述在Ubuntu系统上的安装和使用，Mac的操作应该也差不多。

使用pip安装

使用pip安装非常简单，只要一条命令就够了，如下：

1pip3 install --upgrade visualdl

测试一下是否安装成功了，运行一个例子下载日志文件：

1# 在当前位置下载一个日志
2vdl_create_scratch_log

然后再输入，启动VisualDL并加载这个日志信息：

1visualdl --logdir=scratch_log/ --port=8080

这里说明一下，visualDL的参数：

  • host 设定IP
  • port 设定端口
  • model_pb 指定 ONNX 格式的模型文件，这木方我们还没要用到

注意： 如果是报以下的错误，那是因为protobuf版本过低的原因。

1pip3 install protobuf -U

使用源码安装

如果读者出于各种情况，使用pip安装不能满足需求，那可以考虑使用源码安装VisualDL，操作如下：
首先要安装依赖库：

1# 安装npm
2apt install npm
3# 安装node
4apt install nodejs-legacy
5# 安装cmake
6apt install cmake
7# 安装unzip
8apt install unzip

然后在GitHub上clone最新的源码并打开：

1git clone https://github.com/PaddlePaddle/VisualDL.git
2cd VisualDL

之后是编译生成whl安装包：

1python3 setup.py bdist_wheel

生成whl安装包之后，就可以使用pip命令安装这个安装包了，*号对应的是visualdl版本号，读者要根据实际情况来安装：

1pip3 install --upgrade dist/visualdl-*.whl

安装完成之后，同样可以使用在上一部分的使用pip安装的测试方法测试安装是否成功。

简单使用VisualDL

我们编写下面这一小段的代码来学习VisualDL的使用，test_visualdl.py的代码如下：

 1# 导入VisualDL的包
 2from visualdl import LogWriter
 3
 4# 创建一个LogWriter，第一个参数是指定存放数据的路径，
 5# 第二个参数是指定多少次写操作执行一次内存到磁盘的数据持久化
 6logw = LogWriter("./random_log", sync_cycle=10000)
 7
 8# 创建训练和测试的scalar图，
 9# mode是标注线条的名称，
10# scalar标注的是指定这个组件的tag
11with logw.mode('train') as logger:
12    scalar0 = logger.scalar("scratch/scalar")
13
14with logw.mode('test') as logger:
15    scalar1 = logger.scalar("scratch/scalar")
16
17# 读取数据
18for step in range(1000):
19    scalar0.add_record(step, step * 1. / 1000)
20    scalar1.add_record(step, 1. - step * 1. / 1000)

运行Python代码之后，在终端上输入，从上面的代码可以看到我们定义的路径是./random_log：

1visualdl --logdir=random_log/ --port=8080

然后在浏览器上输入：

1http://127.0.0.1:8080

然后就可以看到刚才编写Python代码生成的图像了：

经过这个例子，读者对VisualDL有了进一步的了解了，那么在接下来的我们就在实际的PaddlePaddle例子中使用我们的VisualDL。

在PaddlePaddle使用VisualDL

下面就介绍在PaddlePaddle训练中使用VisualDL，通过在训练的时候使用VisualDL不断收集训练的数据集，最终通过可视化展示出来。

定义MobileNet V2神经网络

创建一个mobilenet_v2.py来定义一个MobileNet V2神经网络。MobileNet V2是MobileNet V1的升级版，从名字可以看出这个网络是为例移动设备而诞生的，它最大的特点就是模型小，预测速度快，适合部署在移动设备上。MobileNet V2是将MobileNet V1和残差网络ResNet的残差单元结合起来，用Depthwise Convolutions代替残差单元的bottleneck，最重要的是与residuals block相反，通常的residuals block是先经过1×1的卷积，降低feature map通道数，然后再通过3×3卷积，最后重新经过1×1卷积将feature map通道数扩张回去；而且为了避免ReLU对特征的破坏，用线性层替换channel数较少层后的ReLU非线性激活。

  1import paddle.fluid as fluid
  2
  3def conv_bn_layer(input, filter_size, num_filters, stride, padding, num_groups=1, if_act=True, use_cudnn=True):
  4    conv = fluid.layers.conv2d(input=input,
  5                               num_filters=num_filters,
  6                               filter_size=filter_size,
  7                               stride=stride,
  8                               padding=padding,
  9                               groups=num_groups,
 10                               use_cudnn=use_cudnn,
 11                               bias_attr=False)
 12    bn = fluid.layers.batch_norm(input=conv)
 13    if if_act:
 14        return fluid.layers.relu6(bn)
 15    else:
 16        return bn
 17
 18
 19def shortcut(input, data_residual):
 20    return fluid.layers.elementwise_add(input, data_residual)
 21
 22
 23def inverted_residual_unit(input,
 24                           num_in_filter,
 25                           num_filters,
 26                           ifshortcut,
 27                           stride,
 28                           filter_size,
 29                           padding,
 30                           expansion_factor):
 31    num_expfilter = int(round(num_in_filter * expansion_factor))
 32
 33    channel_expand = conv_bn_layer(input=input,
 34                                   num_filters=num_expfilter,
 35                                   filter_size=1,
 36                                   stride=1,
 37                                   padding=0,
 38                                   num_groups=1,
 39                                   if_act=True)
 40
 41    bottleneck_conv = conv_bn_layer(input=channel_expand,
 42                                    num_filters=num_expfilter,
 43                                    filter_size=filter_size,
 44                                    stride=stride,
 45                                    padding=padding,
 46                                    num_groups=num_expfilter,
 47                                    if_act=True,
 48                                    use_cudnn=False)
 49
 50    linear_out = conv_bn_layer(input=bottleneck_conv,
 51                               num_filters=num_filters,
 52                               filter_size=1,
 53                               stride=1,
 54                               padding=0,
 55                               num_groups=1,
 56                               if_act=False)
 57    if ifshortcut:
 58        out = shortcut(input=input, data_residual=linear_out)
 59        return out
 60    else:
 61        return linear_out
 62
 63def invresi_blocks(input, in_c, t, c, n, s, name=None):
 64    first_block = inverted_residual_unit(input=input,
 65                                         num_in_filter=in_c,
 66                                         num_filters=c,
 67                                         ifshortcut=False,
 68                                         stride=s,
 69                                         filter_size=3,
 70                                         padding=1,
 71                                         expansion_factor=t)
 72
 73    last_residual_block = first_block
 74    last_c = c
 75
 76    for i in range(1, n):
 77        last_residual_block = inverted_residual_unit(input=last_residual_block,
 78                                                     num_in_filter=last_c,
 79                                                     num_filters=c,
 80                                                     ifshortcut=True,
 81                                                     stride=1,
 82                                                     filter_size=3,
 83                                                     padding=1,
 84                                                     expansion_factor=t)
 85    return last_residual_block
 86
 87
 88def net(input, class_dim, scale=1.0):
 89    bottleneck_params_list = [
 90        (1, 16, 1, 1),
 91        (6, 24, 2, 2),
 92        (6, 32, 3, 2),
 93        (6, 64, 4, 2),
 94        (6, 96, 3, 1),
 95        (6, 160, 3, 2),
 96        (6, 320, 1, 1),
 97    ]
 98
 99    # conv1
100    input = conv_bn_layer(input,
101                          num_filters=int(32 * scale),
102                          filter_size=3,
103                          stride=2,
104                          padding=1,
105                          if_act=True)
106
107    # bottleneck sequences
108    i = 1
109    in_c = int(32 * scale)
110    for layer_setting in bottleneck_params_list:
111        t, c, n, s = layer_setting
112        i += 1
113        input = invresi_blocks(input=input,
114                               in_c=in_c,
115                               t=t,
116                               c=int(c * scale),
117                               n=n,
118                               s=s,
119                               name='conv' + str(i))
120        in_c = int(c * scale)
121    # last_conv
122    input = conv_bn_layer(input=input,
123                          num_filters=int(1280 * scale) if scale > 1.0 else 1280,
124                          filter_size=1,
125                          stride=1,
126                          padding=0,
127                          if_act=True)
128
129    feature = fluid.layers.pool2d(input=input,
130                                  pool_size=7,
131                                  pool_stride=1,
132                                  pool_type='avg',
133                                  global_pooling=True)
134    net = fluid.layers.fc(input=feature,
135                          size=class_dim,
136                          act='softmax')
137    return net

创建一个train.py开始训练。首先导入相关的依赖包。

1import paddle as paddle
2import paddle.dataset.cifar as cifar
3import paddle.fluid as fluid
4import mobilenet_v2
5from visualdl import LogWriter

创建VisualDL的记录器，通过这个记录器可以记录每次训练的数据，并存储在log/目录下。

 1# 创建记录器
 2log_writer = LogWriter(dir='log/', sync_cycle=10)
 3
 4# 创建训练和测试记录数据工具
 5with log_writer.mode('train') as writer:
 6    train_cost_writer = writer.scalar('cost')
 7    train_acc_writer = writer.scalar('accuracy')
 8    histogram = writer.histogram('histogram', num_buckets=50)
 9
10with log_writer.mode('test') as writer:
11    test_cost_writer = writer.scalar('cost')
12    test_acc_writer = writer.scalar('accuracy')

这里是定义一系列的操作，如定义输入层，获取MobileNet V2的分类器，克隆预测程序，定义优化方法。

 1# 定义输入层
 2image = fluid.layers.data(name='image', shape=[3, 32, 32], dtype='float32')
 3label = fluid.layers.data(name='label', shape=[1], dtype='int64')
 4
 5# 获取分类器
 6model = mobilenet_v2.net(image, 10)
 7
 8# 获取损失函数和准确率函数
 9cost = fluid.layers.cross_entropy(input=model, label=label)
10avg_cost = fluid.layers.mean(cost)
11acc = fluid.layers.accuracy(input=model, label=label)
12
13# 获取训练和测试程序
14test_program = fluid.default_main_program().clone(for_test=True)
15
16# 定义优化方法
17optimizer = fluid.optimizer.AdamOptimizer(learning_rate=1e-3)
18opts = optimizer.minimize(avg_cost)

获取CIFAR的训练数据和测试数据，并创建一个执行器，MobileNet V2这个模型虽然使用在手机上的，但是在训练起来却不是那么快，最好使用GPU进行训练，要不是相当的慢。

 1# 获取CIFAR数据
 2train_reader = paddle.batch(cifar.train10(), batch_size=32)
 3test_reader = paddle.batch(cifar.test10(), batch_size=32)
 4
 5# 定义一个使用CPU的执行器
 6place = fluid.CUDAPlace(0)
 7# place = fluid.CPUPlace()
 8exe = fluid.Executor(place)
 9# 进行参数初始化
10exe.run(fluid.default_startup_program())
11
12# 定义输入数据维度
13feeder = fluid.DataFeeder(place=place, feed_list=[image, label])

这里从初始化程序中获取全部参数的名称，用于之后训练过程中输出参数的值，并记录到VisualDL中。

1# 定义日志的开始位置和获取参数名称
2train_step = 0
3test_step = 0
4params_name = fluid.default_startup_program().global_block().all_parameters()[0].name

开始训练模型，在训练过程中，把训练时的损失值保存到train_cost_writer中，把训练时的准确率保存到train_acc_writer中，把训练过程中的参数变化保存到histogram中。把测试时的损失值保存到test_cost_writer中，把测试时的准确率保存到test_acc_writer中。

 1# 训练10次
 2for pass_id in range(10):
 3    # 进行训练
 4    for batch_id, data in enumerate(train_reader()):
 5        train_cost, train_acc, params = exe.run(program=fluid.default_main_program(),
 6                                                feed=feeder.feed(data),
 7                                                fetch_list=[avg_cost, acc, params_name])
 8        # 保存训练的日志数据
 9        train_step += 1
10        train_cost_writer.add_record(train_step, train_cost[0])
11        train_acc_writer.add_record(train_step, train_acc[0])
12        histogram.add_record(train_step, params.flatten())
13
14        # 每100个batch打印一次信息
15        if batch_id % 100 == 0:
16            print('Pass:%d, Batch:%d, Cost:%0.5f, Accuracy:%0.5f' %
17                  (pass_id, batch_id, train_cost[0], train_acc[0]))
18
19    # 进行测试
20    test_accs = []
21    test_costs = []
22    for batch_id, data in enumerate(test_reader()):
23        test_cost, test_acc = exe.run(program=test_program,
24                                      feed=feeder.feed(data),
25                                      fetch_list=[avg_cost, acc])
26        # 保存测试的日志数据
27        test_step += 1
28        test_cost_writer.add_record(test_step, test_cost[0])
29        test_acc_writer.add_record(test_step, test_acc[0])
30
31        test_accs.append(test_acc[0])
32        test_costs.append(test_cost[0])
33    # 求测试结果的平均值
34    test_cost = (sum(test_costs) / len(test_costs))
35    test_acc = (sum(test_accs) / len(test_accs))
36    print('Test:%d, Cost:%0.5f, Accuracy:%0.5f' % (pass_id, test_cost, test_acc))
