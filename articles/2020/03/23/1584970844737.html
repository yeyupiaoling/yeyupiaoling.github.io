《我的PaddlePaddle学习之路》笔记五——验证码的识别

2020-03-23

*本篇文章基于 PaddlePaddle 0.11.0、Python 2.7

数据集介绍

本次使用的验证码是方正系统，现在很多的大学的教务系统用的就是这个方正系统，刚好既然那么普遍，我们就用它练一练手。经过观察大量的验证码发现，该系统的验证码只有小写的字母和数字，这样分类就少了很多了。该系统的验证码如下：

验证码 尺寸 说明
72*27 只有数字和小写字母
12*27 第一个X方向的从5开始裁剪到17，Y取全部，即从0裁剪到27
12*27 第二个X方向的从17开始裁剪到29，Y取全部，即从0裁剪到27
12*27 第三个X方向的从29开始裁剪到41，Y取全部，即从0裁剪到27
12*27 第四个X方向的从41开始裁剪到53，Y取全部，即从0裁剪到27

通过上面表上说明，我们就可以开始裁剪验证码了。在裁剪之前我们先要下载编写一个程序，让它来帮我们下载更多的验证码

下载验证码

编写一个下载验证码的程序DownloadYanZhengMa.py，这里我们只需要传入保存路径和要下载的数量就可以了。

修改验证码的文件名

从上一个部分可以看到下载到images/download_yanzhengma文件夹中, 待下载完成之后,需要做以下几件事:

  1. 将每一张验证码命名为其对应的验证码内容,这是一个庞大的工作量
  2. 将命名好的验证码剪切到images/src_yanzhengma/文件夹中

修改验证码的文件名是一个非常费时的工程,如果快速正确命名,那要发挥你们的想象力了,笔者同时也提供了数据集,这个不用担心. 正确命名是非常重要的, 在一个部分会讲到.

裁剪验证码

在数据集介绍的那部分，我们编写一个CorpYanZhengMa.py程序，让它来帮我们去裁剪所有的验证码，但是要注意一下几点：

  • 验证码的命名一定要对于验证码的内容，这个非常重要
  • 裁剪的验证码会单独存放在自己对应的文件夹中

生成图像列表

编写一个生成CreateDataList.py的程序，然后我们要把刚才的验证码生成一个图像列表，只有这个这里PaddlePaddle才能读取验证码数据，在自定义图像数据集的识别这一章有介绍，如果不了解的话可以阅读该文章。
这里就用到了上一个部分裁剪后的数据集,通过传入../images/dst_yanzhengma这个路径,会把之前裁剪好的所有图像都生成它的相对路径,给之后的训练程序使用.

 4
 5class CreateDataList:

19        father_paths = data_root_path.split('/')
20        while True:

24                break

26

57                        f.write(name_path + "\t%d" % class_label + "\n")
58                class_sum += 1
59                all_class_images += 1
60            class_label += 1
61            # 说明的json文件的class_detail数据
62            class_detail_list['class_name'] = class_dir
63            class_detail_list['class_label'] = class_label
64            class_detail_list['class_test_images'] = test_sum
65            class_detail_list['class_trainer_images'] = trainer_sum
66            class_detail.append(class_detail_list)
67        # 获取类别数量
68        all_class_sum = class_dirs.__len__()
69        # 说明的json文件信息
70        readjson = {}
71        readjson['all_class_name'] = father_path

75        jsons = json.dumps(readjson, sort_keys=True, indent=4, separators=(',', ': '))

MyReader代码

为了做一下区分,我把命名改成了MyReade.py，在旧版本的该程序是有bug的，如果读者想使用这个程序，想要更新本地PaddlePaddle的版本，旧版本的bug是没有对灰度的图像处理，所以在做这个灰度的验证码时会报错。

 1# coding=utf-8
 2from multiprocessing import cpu_count
 3import paddle.v2 as paddle
 4
 5
 6class MyReader:
 7    def __init__(self, imageSize):
 8        self.imageSize = imageSize
 9
10    def train_mapper(self, sample):
11        img, label = sample
12        # 我这里使用的是本地的image,如果你的paddlepaddle是最新的,也可以使用padd.v2.image
13        # 因为是灰度图,所以is_color=False
14        img = paddle.image.load_image(img, is_color=False)
15        img = paddle.image.simple_transform(img, 38, self.imageSize, True, is_color=False)
16        return img.flatten().astype('float32'), label
17
18    def test_mapper(self, sample):
19        img, label = sample
20        # 我这里使用的是本地的image,如果你的paddlepaddle是最新的,也可以使用padd.v2.image
21        # 因为是灰度图,所以is_color=False
22        img = paddle.image.load_image(img, is_color=False)
23        img = paddle.image.simple_transform(img, 38, self.imageSize, False, is_color=False)
24        return img.flatten().astype('float32'), label
25
26    def train_reader(self, train_list, buffered_size=1024):
27        def reader():
28            with open(train_list, 'r') as f:
29                lines = [line.strip() for line in f]
30                for line in lines:
31                    img_path, lab = line.strip().split('\t')
32                    yield img_path, int(lab)
33
34        return paddle.reader.xmap_readers(self.train_mapper, reader,
35                                          cpu_count(), buffered_size)
36
37    def test_reader(self, test_list, buffered_size=1024):
38        def reader():
39            with open(test_list, 'r') as f:
40                lines = [line.strip() for line in f]
41                for line in lines:
42                    img_path, lab = line.strip().split('\t')
43                    yield img_path, int(lab)
44
45        return paddle.reader.xmap_readers(self.test_mapper, reader,
46                                          cpu_count(), buffered_size)

使用PaddlePaddle开始训练

同样在开始训练的前，要定义一个神经网络vgg.py，我们这次使用的还是前面两章使用到的VGG16的神经网络模型,在这里我同样是把BN层关闭了,至于为什么,可以查看之前的文章：

 1# coding:utf-8
 2import paddle.v2 as paddle
 3
 4
 5# ***********************定义VGG卷积神经网络模型***************************************
 6def vgg_bn_drop(datadim, type_size):
 7    image = paddle.layer.data(name="image",
 8                              type=paddle.data_type.dense_vector(datadim))
 9
10    def conv_block(ipt, num_filter, groups, dropouts, num_channels=None):
11        return paddle.networks.img_conv_group(
12            input=ipt,
13            num_channels=num_channels,
14            pool_size=2,
15            pool_stride=2,
16            conv_num_filter=[num_filter] * groups,
17            conv_filter_size=3,
18            conv_act=paddle.activation.Relu(),
19            conv_with_batchnorm=False,
20            conv_batchnorm_drop_rate=dropouts,
21            pool_type=paddle.pooling.Max())
22
23    # 最后一个参数是图像的通道数
24    conv1 = conv_block(image, 64, 2, [0.3, 0], 1)
25    conv2 = conv_block(conv1, 128, 2, [0.4, 0])
26    conv3 = conv_block(conv2, 256, 3, [0.4, 0.4, 0])
27    conv4 = conv_block(conv3, 512, 3, [0.4, 0.4, 0])
28    conv5 = conv_block(conv4, 512, 3, [0.4, 0.4, 0])
29
30    drop = paddle.layer.dropout(input=conv5, dropout_rate=0.5)
31    fc1 = paddle.layer.fc(input=drop, size=512, act=paddle.activation.Linear())
32    bn = paddle.layer.batch_norm(input=fc1,
33                                 act=paddle.activation.Relu(),
34                                 layer_attr=paddle.attr.Extra(drop_rate=0.5))
35    fc2 = paddle.layer.fc(input=bn, size=512, act=paddle.activation.Linear())
36    # 通过Softmax获得分类器
37    out = paddle.layer.fc(input=fc2,
38                          size=type_size,
39                          act=paddle.activation.Softmax())
40    return out

 1def get_parameters(self, parameters_path=None, cost=None):
 2    if not parameters_path:
 3        # 使用cost创建parameters
 4        if not cost:
 5            raise NameError('请输入cost参数')
 6        else:

11    else:

创建训练器

创建训练器要3个参数,分别是损失函数,参数,优化方法.
通过图像的标签信息和分类器生成损失函数.
参数可以选择是使用之前训练好的参数,然后在此基础上再进行训练,又或者是使用损失函数生成初始化参数.
然后再生成优化方法.就可以创建一个训练器了.

 1# datadim 数据大小
 2def get_trainer(self, datadim, type_size, parameters_path):
 3    # 获得图片对于的信息标签
 4    label = paddle.layer.data(name="label",
 5                              type=paddle.data_type.integer_value(type_size))
 6
 7    # 获取全连接层,也就是分类器
 8    out = vgg_bn_drop(datadim=datadim, type_size=type_size)
 9
10    # 获得损失函数
11    cost = paddle.layer.classification_cost(input=out, label=label)
12
13    # 获得参数
14    if not parameters_path:
15        parameters = self.get_parameters(cost=cost)
16    else:
17        parameters = self.get_parameters(parameters_path=parameters_path)
18
19    '''
20    定义优化方法
21    learning_rate 迭代的速度
22    momentum 跟前面动量优化的比例
23    regularzation 正则化,防止过拟合
24    '''
25    # ********************如果使用VGG网络模型就用这个优化方法******************
26    optimizer = paddle.optimizer.Momentum(
27        momentum=0.9,
28        regularization=paddle.optimizer.L2Regularization(rate=0.0005 * 128),
29        learning_rate=0.0001 / 128,
30        learning_rate_decay_a=0.1,
31        learning_rate_decay_b=128000 * 35,
32        learning_rate_schedule="discexp", )
33
34    # ********************如果使用LeNet-5网络模型就用这个优化方法******************
35    # optimizer = paddle.optimizer.Momentum(learning_rate=0.00001 / 128.0,
36    #                                       momentum=0.9,
37    #                                       regularization=paddle.optimizer.L2Regularization(rate=0.005 * 128))
38
39    '''
40    创建训练器
41    cost 分类器
42    parameters 训练参数,可以通过创建,也可以使用之前训练好的参数
43    update_equation 优化方法
44    '''
45    trainer = paddle.trainer.SGD(cost=cost,
46                                 parameters=parameters,
47                                 update_equation=optimizer)
48    return trainer

开始训练

我们把图片设置成灰度图来处理了，所以数据集也非常小。所以相对之前训练CIFAR数据集来说,训练起来还算挺快的。

 1# ***********************开始训练***************************************
 2def start_trainer(self, trainer, num_passes, save_parameters_name, trainer_reader, test_reader):
 3    # 获得数据
 4    reader = paddle.batch(reader=paddle.reader.shuffle(reader=trainer_reader,
 5                                                       buf_size=50000),
 6                          batch_size=128)
 7    # 保证保存模型的目录是存在的
 8    father_path = save_parameters_name[:save_parameters_name.rfind("/")]
 9    if not os.path.exists(father_path):
10        os.makedirs(father_path)
11
12    # 指定每条数据和padd.layer.data的对应关系
13    feeding = {"image": 0, "label": 1}
14
15    # 定义训练事件
16    def event_handler(event):
17        if isinstance(event, paddle.event.EndIteration):
18            if event.batch_id % 100 == 0:
19                print "\nPass %d, Batch %d, Cost %f, %s" % (
20                    event.pass_id, event.batch_id, event.cost, event.metrics)
21            else:
22                sys.stdout.write('.')
23                sys.stdout.flush()
24
25        # 每一轮训练完成之后
26        if isinstance(event, paddle.event.EndPass):
27            # 保存训练好的参数
28            with open(save_parameters_name, 'w') as f:
29                trainer.save_parameter_to_tar(f)
30
31            # 测试准确率
32            result = trainer.test(reader=paddle.batch(reader=test_reader,
33                                                      batch_size=128),
34                                  feeding=feeding)
35            print "\nTest with Pass %d, %s" % (event.pass_id, result.metrics)
36
37    '''
38    开始训练
39    reader 训练数据
40    num_passes 训练的轮数
41    event_handler 训练的事件,比如在训练的时候要做一些什么事情
42    feeding 说明每条数据和padd.layer.data的对应关系
43    '''
44    trainer.train(reader=reader,
45                  num_passes=num_passes,
46                  event_handler=event_handler,
47                  feeding=feeding)

使用PaddlePaddle预测

编写infer.py做验证码预测，这次预测要做的事情比较多.
因为传进来的是一个完整的验证码,所以首先要对验证码进行裁剪.
然后把裁剪后的数据传该PaddlePaddle进行预测.
预测出来的是一个label值,所以还有通过label找到对应的字符

裁剪要预测的验证码

使用裁剪好的图像进行预测

 1def to_prediction(test_data, parameters, out, all_class_name):
 2    with open('../data/%s/readme.json' % all_class_name) as f:
 3        txt = f.read()
 4    # 获得预测结果
 5    probs = paddle.infer(output_layer=out,
 6                         parameters=parameters,
 7                         input=test_data)
 8    # 处理预测结果
 9    lab = np.argsort(-probs)
10    # 返回概率最大的值和其对应的概率值
11    result = ''
12    for i in range(0, lab.__len__()):
13        print '第%d张预测结果为:%d,可信度为:%f' % (i + 1, lab[i][0], probs[i][(lab[i][0])])
14        result = result + lab_to_result(lab[i][0], txt)
15    return str(result)

把预测的label转换成对应的字符

1def lab_to_result(lab, json_str):
2    myjson = json.loads(json_str)
3    class_details = myjson['class_detail']
4    for class_detail in class_details:
5        if class_detail['class_label'] == lab:
6            return class_detail['class_name']
