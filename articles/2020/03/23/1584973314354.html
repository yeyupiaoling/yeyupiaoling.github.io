前言

本文章我们来学习一下使用PaddlePaddle实现人脸对比和人脸识别，使用的训练数据集是CASIA-WebFace。

数据集介绍

我们使用的是CASIA-WebFace数据集，该人脸数据集是目前最大的公开人脸数据集。该人脸数据集一共有包含10,575个人，494,414张图像，包含彩色图和灰图。各大人脸数据集情况如下表。

CASIA-WebFace 10,575 494,414 Public

编写读取图像的reader，这个reader对图像做的预处理的进行中心裁剪，因为人脸都是居中的，进行居中裁剪可以去掉其他的背景的影响。

 1# 把图像和label读取成reader
 2# coding=utf-8
 3import cv2
 4import numpy as np
 5import paddle.v2 as paddle
 6import random
 7from multiprocessing import cpu_count

 9class MyReader:
10    def __init__(self, imageSize, type_size, center_crop_size = 128):
11        self.imageSize = imageSize
12        self.type_size = type_size
13        self.center_crop_size = center_crop_size
14        self.default_image_size = 250
15
16    def train_mapper(self, sample):
17        img, label = sample
18        sparse_label = [0 for i in range(self.type_size)]
19        sparse_label[label - 1] = 1
20        # 裁剪中心图片
21        def crop_img(img, center_crop_size):
22            img = cv2.imread(img, 0)
23            if center_crop_size < self.default_image_size:
24                side = (self.default_image_size - center_crop_size) / 2
25                img = img[side: self.default_image_size - side - 1, side: self.default_image_size - side - 1]
26            return img
27
28        img = crop_img(img, self.center_crop_size)
29        img = cv2.resize(img, (self.imageSize, self.imageSize))
30        return img.flatten().astype('float32'), label, sparse_label
31    # 获取训练的reader
32    def train_reader(self, train_list, buffered_size=1024):
33        def reader():
34            with open(train_list, 'r') as f:
35                lines = [line.strip() for line in f]
36                # 打乱数据
37                random.shuffle(lines)
38                for line in lines:
39                    line = line.strip().split('\t')
40                    img_path = line[0]
41                    img_label = line[1]
42
43                    yield img_path, int(img_label)
44
45        return paddle.reader.xmap_readers(self.train_mapper, reader, cpu_count(), buffered_size)

编写卷积神经网络，这个是根据resnet修改的网络。使用了6个卷积块，最后的返回值是最后一个池化层和最后一个全连接层，输出最后一层池化层是为了在预测的是获取图像的人脸特征，做人脸对比。

 1import numpy as np
 2import paddle.v2 as paddle
 3
 4
 5def conv_bn_layer(input,
 6                  ch_out,
 7                  filter_size,
 8                  stride,
 9                  padding,
10                  active_type=paddle.activation.Relu(),
11                  ch_in=None):
12    tmp = paddle.layer.img_conv(
13        input=input,
14        filter_size=filter_size,
15        num_channels=ch_in,
16        num_filters=ch_out,
17        stride=stride,
18        padding=padding,
19        act=paddle.activation.Linear(),
20        bias_attr=False)
21    return paddle.layer.batch_norm(input=tmp, act=active_type, moving_average_fraction=0.999)
22
23
24def shortcut(ipt, ch_in, ch_out, stride):
25    if ch_in != ch_out:
26        return conv_bn_layer(ipt, ch_out, 1, stride, 0, paddle.activation.Linear())
27    else:
28        return ipt
29
30def basicblock(ipt, ch_in, ch_out, stride):
31    tmp = conv_bn_layer(ipt, ch_out, 3, stride, 1)
32    tmp = conv_bn_layer(tmp, ch_out, 3, 1, 1, paddle.activation.Linear())
33    short = shortcut(ipt, ch_in, ch_out, stride)
34    return paddle.layer.addto(input=[tmp, short], act=paddle.activation.Relu())
35
36
37def layer_warp(block_func, ipt, ch_in, ch_out, count, stride):
38    tmp = block_func(ipt, ch_in, ch_out, stride)
39    for i in range(1, count):
40        tmp = block_func(tmp, ch_out, ch_out, 1)
41    return tmp
42
43
44def resnet(ipt, class_dim):
45    n = 1
46    feature_maps = 512
47    ipt_bn = ipt - 128.0
48    # 获取卷积层输出
49    conv1 = conv_bn_layer(ipt_bn, ch_in=1, ch_out=8, filter_size=3, stride=1, padding=1)
50    # 多个残差块组合
51    res0 = layer_warp(basicblock, conv1, 8, 16, n, 1)
52    res1 = layer_warp(basicblock, res0, 16, 32, n, 1)
53    res2 = layer_warp(basicblock, res1, 32, 64, n, 2)
54    res3 = layer_warp(basicblock, res2, 64, 128, n, 2)
55    res4 = layer_warp(basicblock, res3, 128, 256, n, 2)
56    res5 = layer_warp(basicblock, res4, 256, feature_maps, n, 2)
57    # 最后使用池化层来降维
58    pool = paddle.layer.img_pool(input=res5, name='pool', pool_size=8, stride=1, pool_type=paddle.pooling.Avg())
59
60    fc = paddle.layer.fc(input=pool, size=class_dim, act=paddle.activation.Softmax())
61    return pool, fc

预测

经过上面的训练之后，获得得到了一个训练好的模型，我们将会使用这个模型来进行人脸对比和人脸识别。

人脸对比

人脸对比，人脸对比其实就是做普通的分类预测，但是输出的不是最后一层全连接层，而是最后一层池化层，这样输出的就是人脸的特征，然后使用对角余弦函数来计算他们的相似度。

  • 通过人脸对比的方式实现一些场景的应用。比如对比证件上的人脸和真实的人脸是否为同一个人，操作方式判断人脸相似度的result是否达到预设值，推荐相似度为0.8时，为同一个人。
  • 利用这种的人脸对比方式，有可以实现人脸识别。
    • 首先我们可以把人脸以注册人脸的方式加入到注册人脸库中，加关联到该人脸的信息；
    • 然后要进行识别时，把要识别的人脸和已注册的人脸库中的人脸进行对比，当对比为识别为同一个人脸，就算识别成功
    • 这样的处理方式好处是，不需要每次增加新的用户时，需要收集大量该用户的人脸，只有收集一张或者多张多角度的人脸，完全可以使用同一个模型进行人脸对比。

人脸识别

这个是人脸识别方式是不推荐使用的，它就是一个分类的操作，输入一张人脸图片，获取对应的人脸的label和概率。
但是如果要加入新的人脸，需要收集大量该用户的人脸，并再次进行训练，得到新的模型。
这样的识别方式，扩展性非常弱，但是识别速度比较快，不需要每张人脸都进行对比。
