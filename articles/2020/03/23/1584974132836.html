均值（mean，average）

  • 代表一组数据在分布上的集中趋势和总体上的平均水平。
  • 常说的中心化（Zero-Centered）或者零均值化（Mean-subtraction），就是把每个数值都减去均值。
\mu=\frac{1}{N}\sum_{i=1}^Nx_i\left(x:x_1,x_2,...,x_N\right)
1import numpy as np
2
3# 一维数组
4x = np.array([-0.02964322, -0.11363636, 0.39417967, -0.06916996, 0.14260276])
5print('数据：', x)
6
7# 求均值
8avg = np.mean(x)
9print('均值：', avg)

输出：

1数据： [-0.02964322 -0.11363636  0.39417967 -0.06916996  0.14260276]
2均值： 0.064866578

标准差（Standard Deviation）

  • 代表一组数据在分布上第离散程度。
  • 方差是标准差的平方。
\sigma=\sqrt{\frac{1}{N}\sum_{i=1}^N\left(x_i-\mu\right)^2} \left(x:x_1,x_2,...,x_N\right)
1import numpy as np
2
3# 一维数组
4x = np.array([-0.02964322, -0.11363636, 0.39417967, -0.06916996, 0.14260276])
5print('数据：', x)
6
7# 求标准差
8std = np.std(x)
9print('标准差：', std)

输出：

1数据： [-0.02964322 -0.11363636  0.39417967 -0.06916996  0.14260276]
2标准差： 0.18614576055671836

正态分布（Normal Distribution）

  • 又叫“常态分布”，“高斯分布”，是最重要的一种分布。
  • 均值决定位置，方差决定幅度。
  • 表示：X\sim N\left(\mu,\sigma^2\right)

正态分布的概率密度函数：

f(x)=\frac{1}{\sqrt {2\pi\sigma}}e^{-\frac{\left(x-\mu\right)^2}{2\sigma^2}}

Softmax函数

  • 在分类网络的输出层中作激活函数，输出概率；
  • 先通过指数函数，把所有输出都变为整数；
  • 再做归一化，每个数都除以所有数的和，输出各分类的概率。
P_i=\frac{e^{x_i}}{\sum_{i=1}^Ne^{x_i}}\left(x:x_1,x_2,\ldots,x_N\right)
1import numpy as np
2
3x = np.array([-0.02964322, -0.11363636, 0.39417967, -0.06916996, 0.14260276])
4print('原始输出：', x)
5prob = np.exp(x)/np.sum(np.exp(x))
6print('概率输出：', prob)

输出：

1原始输出： [-0.02964322 -0.11363636  0.39417967 -0.06916996  0.14260276]
2概率输出： [0.17868493 0.16428964 0.27299323 0.17175986 0.21227234]

One-hot 编码

  • 在分类网络中，用于对类别进行编码；
  • 编码长度等于类别数；
  • 每个编码只有一位为1，其余都为0；
  • 所有编码向量都正交。

示例：
假如有5类，则编码为：
第一类：[1, 0, 0, 0, 0]
第二类：[0, 1, 0, 0, 0]
第三类：[0, 0, 1, 0, 0]
第四类：[0, 0, 0, 1, 0]
第五类：[0, 0, 0, 0, 1]

交叉熵（Cross Entropy）

  • 常用于分类问题中，做损失函数；
  • 从分布的角度，让预测概率趋近于标签。

Label:[l_1, l_2, \ldots, l_N] ——经过One-hot编码

predict:[P_1, P_2, \ldots, P_N] ——经过Softmax函数

ce=-\sum_{i=1}^Nl_i\cdot\log^{P_i}
 1import numpy as np
 2
 3# one-hot 编码的标签
 4label = np.array([0,0,1,0,0])
 5print('分类标签：', label)
 6# 网络实际输出
 7x1 = np.array([-0.02964322, -0.11363636, 3.39417967, -0.06916996, 0.14260276])
 8x2 = np.array([-0.02964322, -0.11363636, 1.39417967, -0.06916996, 5.14260276])
 9print('网络输出1:', x1)
10print('网络输出2:', x2)
11# softmax 之后的模拟概率
12p1 = np.exp(x1) / np.sum(np.exp(x1))
13p2 = np.exp(x2) / np.sum(np.exp(x2))
14print('概率输出1:', p1)
15print('概率输出2:', p2)
16# 交叉熵
17ce1 = -np.sum(label * np.log(p1))
18ce2 = -np.sum(label * np.log(p2))
19print('交叉熵1:', ce1)
20print('交叉熵2:', ce2)

输出：

1分类标签： [0 0 1 0 0]
2网络输出1: [-0.02964322 -0.11363636  3.39417967 -0.06916996  0.14260276]
3网络输出2: [-0.02964322 -0.11363636  1.39417967 -0.06916996  5.14260276]
4概率输出1: [0.02877271 0.02645471 0.88293386 0.0276576  0.03418112]
5概率输出2: [0.00545423 0.00501482 0.02265122 0.00524284 0.96163688]
6交叉熵1: 0.12450498821197674
7交叉熵2: 3.787541448750617

激活函数（Activation Function）

  • 引入非线性因素，使模型有更强的表达能力；
  • 输出层采用softmax激活，可以模拟输出概率；
  • sigmoid和tanh都有饱和区，会导致梯度消失；
  • 在深度学习中，sigmoid和tanh主要用于做各种门或开关；
  • 在深度学习中，最常用的激活函数为ReLU及其变体。
\delta(x)=\frac{1}{1+e^{-x}}
Tanh(x)=\frac{e^x-e^{-x}}{e^x+e^{-x}}
ReLU(x)=max(x, 0)
 1import numpy as np
 2from matplotlib import pyplot as plt
 3
 4x = np.linspace(-10, 10, 100)
 5# plt.figure(31)
 6plt.figure(figsize=(10, 20))

 8# Sigmoid
 9sigmoid = 1 / (1 + np.exp(-x))
10top = np.ones(100)

15
16# Tanh
17tanh = (np.exp(x) - np.exp(-x)) / (np.exp(x) + np.exp(-x))
18top = np.ones(100)

24plt.title('Tanh')
25
26# ReLU
27relu = np.maximum(x, 0)
