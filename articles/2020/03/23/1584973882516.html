前言

开发人脸识别系统，人脸数据集是必须的。所以在我们开发这套人脸识别系统的准备工作就是获取人脸数据集。本章将从公开的数据集到自制人脸数据集介绍，为我们之后开发人脸识别系统做好准备。

公开人脸数据集

公开的人脸数据集有很多，本中我们就介绍几个比较常用的人脸数据集。

CelebA人脸数据集

官方提供的下载地址：https://pan.baidu.com/s/1eSNpdRG#list/path=%2F

该数据集下载后有3个文件夹，Anno文件夹是存放标注文件的，Eval文件夹是存放评估列表文件的，Img文件是存放图片文件的。
Img中有3中类型的图像文件，其中

  • img_align_celeba.zip是经过对人脸居中，裁剪，并统一大小为178*178的jpg图片；

  • img_celeba.7z这个是人脸图片的原始图片，没有经过居中裁剪等处理的图片。

制作人脸数据集

下面我们就介绍如何制作自己的人脸数据集，项目的开源地址：https://github.com/yeyupiaoling/FaceDataset 。该项目可以分为两个阶段，第一阶段是人脸图片的获取和简单的清洗，第二阶段是人脸图片的高级清洗和标注人脸信息。人脸信息的标注和清洗使用到了百度的人脸识别服务。

第一阶段

爬取人脸图片的核心思路就是获取中国明星的名字，然后使用明星的名字作为图片搜索的关键字进行获取图片，然后删除下载过程损坏的图片和没有包含人脸的图片，或者过多人脸的图片（我们只保存一张图片只包含一张人脸的图片）。

首先获取中国明星的名字，该功能主要在get_star_name.py中实现。获取明显的名字核心代码如下，获取的名字不能保证百分之百正确，所以可能需要手动去检查。

然后根据明星的名字从网上下载图片，该功能主要在download_image.py中实现，以下就是下载图片的核心代码片段。

 1# 获取百度图片下载图片
 2def download_image(key_word, download_max):
 3    download_sum = 0
 4    str_gsm = '80'
 5    # 把每个明显的图片存放在单独一个文件夹中
 6    save_path = 'star_image' + '/' + key_word
 7    if not os.path.exists(save_path):
 8        os.makedirs(save_path)
 9    while download_sum < download_max:
10        # 下载次数超过指定值就停止下载
11        if download_sum >= download_max:
12            break
13        str_pn = str(download_sum)
14        # 定义百度图片的路径
15        url = 'http://image.baidu.com/search/flip?tn=baiduimage&ie=utf-8&' \
16              'word=' + key_word + '&pn=' + str_pn + '&gsm=' + str_gsm + '&ct=&ic=0&lm=-1&width=0&height=0'
17        try:
18            # 获取当前页面的源码
19            result = requests.get(url, timeout=30).text
20            # 获取当前页面的图片URL
21            img_urls = re.findall('"objURL":"(.*?)",', result, re.S)
22            if len(img_urls) < 1:
23                break
24            # 把这些图片URL一个个下载
25            for img_url in img_urls:
26                # 获取图片内容
27                img = requests.get(img_url, timeout=30)
28                img_name = save_path + '/' + str(uuid.uuid1()) + '.jpg'
29                # 保存图片
30                with open(img_name, 'wb') as f:
31                    f.write(img.content)
32                with open('image_url_list.txt', 'a+', encoding='utf-8') as f:
33                    f.write(img_name + '\t' + img_url + '\n')
34                download_sum += 1
35                if download_sum >= download_max:
36                    break
37        except Exception as e:
38            download_sum += 1
39            continue

下载的图片中可能没有人脸，或者包含多张人脸，所以我们要把这些图片删除掉，该功能主要在delete_more_than_one.py中实现。删除没有人脸或者过多人脸图片的关键代码片段如下。

 1# 删除两个人脸以上的图片或者没有人脸的图片
 2def delete_image(result, image_path):
 3    try:
 4        face_num = int(result['result']['face_num'])
 5        if face_num is not 1:
 6            os.remove(image_path)
 7        else:
 8            face_type = result['result']['face_list'][0]['face_type']['type']
 9            probability = result['result']['face_list'][0]['face_type']['probability']
10            if face_type == 'cartoon' and probability > 0.8:
11                os.remove(image_path)
12    except:
13        os.remove(image_path)

第二阶段

第二阶段属于高级清理和标注人脸信息。这一个阶段首先是把每个文件夹中包含相同一个人的图片较多的人脸，选择其中一个作为主人脸图片。然后使用这个主图片来对比其他图片，判断是否是同一个人，如果不是就删除该图片。接着就删除URL文件中，一些删除的文件对应的URL。最好就使用百度的人脸检测服务标注清理后的图片，最终得到一个人脸数据集。

首先是从众多图片中选择一个主图片，这个功能主要在find_same_person.py中实现，以下是获取主图片的核心代码片段。这个程序消耗时间比较多，其实也可以通过手动标记的方式，选择一个主的人脸图片，当然这个是非常大的一个工作量。

 1# 寻找同一个人的作为0.jpg，作为主的参考图片
 2def find_same_person(person_image_path):
 3    # 获取该人中的所有图片
 4    image_paths = os.listdir(person_image_path)
 5    if '0.jpg' in image_paths:
 6        image_paths.remove('0.jpg')
 7    # 临时选择第一个作为主图片
 8    temp_image = os.path.join(person_image_path, image_paths[0])
 9    main_path = os.path.join(person_image_path, '0.jpg')
10    if os.path.exists(main_path):
11        os.remove(main_path)
12    shutil.copyfile(temp_image, main_path)
13    for main_image in image_paths:
14        # 获取主图片的全路径
15        main_image = os.path.join(person_image_path, main_image)
16        # 获取主图片的base64
17        main_img = get_file_content(main_image)
18        # 统计相同人脸数量
19        same_sum = 0
20        for other_image in image_paths:
21            # 获取其他对比人脸的全路径
22            other_image = os.path.join(person_image_path, other_image)
23            # 获取其他对比图片的base64
24            other_img = get_file_content(other_image)
25            # 获取对比结果
26            result = match_image(main_img, other_img)
27            time.sleep(0.5)
28            # 判断是不是同一个人
29            if if_same_person(result):
30                same_sum += 1
31            # 当相同的人脸超过6个是就做为主图片
32            if same_sum >= 6:
33                if os.path.exists(main_path):
34                    os.remove(main_path)
35                shutil.copyfile(main_image, main_path)
36                break
37        if same_sum > 6:
38            break

最后执行annotate_image.py程序，利用百度人脸检测接口标注人脸图片，以下是标注人脸的核心代码片段。

 1# 把预测结果和图片的URL写入到标注文件中
 2def annotate_image(result, image_path, image_url):
 3    # 获取文件夹名字，并得到已经记录多少人
 4    father_path = os.path.dirname(image_path)
 5    image_name = os.path.basename(image_path).split('.')[0]
 6    # 获取明星的名字
 7    name = father_path.split('/')[-1]
 8    # 把这些名字转换成数字标号
 9    names.add(name)
10    num_name = str(len(names) - 1)
11    annotation_path = os.path.join('annotations', num_name)
12    dict_names_list.append((name, num_name))
13    annotation_file_path = os.path.join(annotation_path, str(image_name) + '.json')
14    # 创建存放标注文件的文件夹
15    if not os.path.exists(annotation_path):
16        os.makedirs(annotation_path)
17
18    try:
19        # 名字
20        name = name
21        # 年龄
22        age = result['result']['face_list'][0]['age']
23        # 性别，male:男性 female:女性
24        gender = result['result']['face_list'][0]['gender']['type']
25        # 脸型，square: 正方形 triangle:三角形 oval: 椭圆 heart: 心形 round: 圆形
26        face_shape = result['result']['face_list'][0]['face_shape']['type']
27        # 是否带眼镜，none:无眼镜，common:普通眼镜，sun:墨镜
28        glasses = result['result']['face_list'][0]['glasses']['type']
29        # 表情，none:不笑；smile:微笑；laugh:大笑
30        expression = result['result']['face_list'][0]['expression']['type']
31        # 颜值，范围0-100
32        beauty = result['result']['face_list'][0]['beauty']
33        # 人脸在图片中的位置
34        location = str(result['result']['face_list'][0]['location']).replace("'", '"')
35        # 人脸旋转角度参数
36        angle = str(result['result']['face_list'][0]['angle']).replace("'", '"')
37        # 72个特征点位置
38        landmark72 = str(result['result']['face_list'][0]['landmark72']).replace("'", '"')
39        # 4个关键点位置，左眼中心、右眼中心、鼻尖、嘴中心
40        landmark = str(result['result']['face_list'][0]['landmark']).replace("'", '"')
41        # 拼接成符合json格式的字符串
42        txt = '{"name":"%s", "image_url":"%s","age":%f, "gender":"%s", "glasses":"%s", "expression":"%s", "beauty":%f, "face_shape":"%s", "location":%s, "angle":%s, "landmark72":%s, "landmark":%s}' \
43              % (name, image_url, age, gender, glasses, expression, beauty, face_shape, location, angle, landmark72,
44                 landmark)
45        # 转换成json数据并格式化
46        json_dicts = json.loads(txt)
47        json_format = json.dumps(json_dicts, sort_keys=True, indent=4, separators=(',', ':'))
48        # 写入标注文件
49        with open(annotation_file_path, 'w', encoding='utf-8') as f_a:
50            f_a.write(json_format)
51    except Exception as e:
52        os.remove(image_path)
53        pass

整个项目完成的时间的非常久的，特别是使用到百度AI服务的程序，为了不出现每秒访问次数超过2次（免费的版本是每秒自动访问2次），所在做了休眠处理，所以这样浪费了不少时间。
