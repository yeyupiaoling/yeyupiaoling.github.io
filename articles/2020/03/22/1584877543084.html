Android读取短信和联系人

2020-03-22
   
Android 夜雨飘零
    0 0

读取短信和联系人经常会用到，要了解的是这是内容提供者（contentProvider）的知识点，大家都知道数据库是在data-->data 对应的包目录下，其他应用是不可以访问到的，如果有些数据要提供给其他的应用，比如通信录，其他应用想获取它的数据怎么办呢，这时内容提供者就出现了，预先写好了一些操作数据库的方法，因为方法是开发这个应用的开发者提供的，所以保证了数据库的安全操作，又能共享数据。

短信

开始切入正题，先说短信，要从数据库中获取数据，就要了解其结构

短信数据库在data-->data中的下面路径

它的表结构如下，有3个是我们要注意的，这3个是我们待会要用到的，其中date是毫秒值

Java代码为

 1　　　　　//获取内容提供者
 2        ContentResolver contentResolver = getContentResolver();
 3        //获取短信表的路径
 4        Uri uri = Uri.parse("content://sms");
 5        //设置要查询的列名
 6        String[] line = {"address", "date", "body"};
 7        //各个参数的意思，路径、列名、条件、条件参数、排序
 8        Cursor cursor = contentResolver.query(uri, line, null, null, null);
 9        //下面就跟操作普通数据库一样了
10        if (cursor != null) {
11            while (cursor.moveToNext()) {
12                String address = cursor.getString(cursor.getColumnIndex("address"));
13                String date = cursor.getString(cursor.getColumnIndex("date"));
14                String body = cursor.getString(cursor.getColumnIndex("body"));
15                Log.e("短信", "address:" + address + "\ndate:" + date + "\nbody:" + body);
16            }
17            cursor.close();
18        }

联系人

获取联系人就比较复杂了，其表结构也比较复杂，我们获取联系人的姓名、电话号码、邮箱，要通过3张表去提取

数据库所在位置data-->data中的下面路径

data表，这个表中的“data1”列几乎包含了我们想要的数据，但是如何通过分类提取，第四列的代号说明了data1数据的类型，这些代号在mimetypes表中有对应的说明，要找到这些数据，又要通过raw_contact_id去查找，这些ID又来于raw_contacts表中

mimetypes表，对代号类型的说明

raw_contacts表，我们通信录中要显示的信息必须要在这里找到对应的ID，当我们删除了联系人，data表中的数据并没有删除，只是删除了证张表contact_id变为null，而不是真的删除

了解其表结构后，就开始写代码了

 1//获取内容提供者
 2        ContentResolver contentResolver = getContentResolver();
 3        //获取raw_contacts表的路径
 4        Uri raw_contact_uri = Uri.parse("content://com.android.contacts/raw_contacts");
 5        //各个参数的意思，路径、列名、条件、条件参数、排序
 6        Cursor contactId = contentResolver.query(raw_contact_uri, new String[]{"contact_id"}, null, null, null);
 7        //下面就跟操作普通数据库一样了
 8        if (contactId != null) {
 9            while (contactId.moveToNext()) {
10                //获取contact_id的值，它对应data表中raw_contact_id的值
11                String id = contactId.getString(contactId.getColumnIndex("contact_id"));
12
13                //获取data表的路径
14                Uri data_uri = Uri.parse("content://com.android.contacts/data");
15                //各个参数的意思，路径、列名、条件、条件参数、排序
16                Cursor dataCursor = contentResolver.query(data_uri, new String[]{"mimetype", "data1"},
17                        "raw_contact_id=?", new String[]{id}, null);
18                if (dataCursor != null) {
19                    //每次循环创建一个实例用于保存data表中的数据
20                    ContactsData contactsData = new ContactsData();
21                    while (dataCursor.moveToNext()) {
22                        String type = dataCursor.getString(dataCursor.getColumnIndex("mimetype"));
23                        switch (type) {
24                            case "vnd.android.cursor.item/email_v2":
25                                //这是邮箱信息
26                                contactsData.setEmail(dataCursor.getString(dataCursor.getColumnIndex("data1")));
27                                break;
28                            case "vnd.android.cursor.item/phone_v2":
29                                //这是手机号码信息
30                                contactsData.setNumber(dataCursor.getString(dataCursor.getColumnIndex("data1")));
31                                break;
32                            case "vnd.android.cursor.item/name":
33                                //这是联系人的名字
34                                contactsData.setName(dataCursor.getString(dataCursor.getColumnIndex("data1")));
35                                break;
36                        }
37                    }
38                    //把查询到的信息添加到集合中
39                    contactsDatas.add(contactsData);
40                    dataCursor.close();
41                }
42            }
43            contactId.close();
44        }
45        Log.d("测试","开始打印");

真正应用

通常不会这样直接写它的路径的，因为手机厂商会修改它的路径，还是这样写的话会报错，说不存在这一列，需要使用Android提供的API

 1　　　　 /**
 2         * 通过ContactsContract.Contacts.CONTENT_URI来获取_ID和DISPLAY_NAME
 3         * _ID 该联系人的索引
 4         * 通过这个ID可以在ContactsContract.CommonDataKinds.Phone.CONTENT_URI 中找到该联系人的电话号码
 5         * 通过这个ID可以在ContactsContract.CommonDataKinds.Email.CONTENT_URI 找到该联系人的邮箱
 6         * DISPLAY_NAME 是该联系人的姓名
 7         */
 8        Cursor cursor = getContentResolver().query(ContactsContract.Contacts.CONTENT_URI, null, null, null, null);
 9        if (cursor != null) {
10            while (cursor.moveToNext()) {
11                ContactsData contactsData = new ContactsData();
12                //获取该联系人的ID
13                String contactId = cursor.getString(cursor.getColumnIndex(ContactsContract.Contacts._ID));
14                //该联系人的姓名
15                String name = cursor.getString(cursor.getColumnIndex(ContactsContract.Contacts.DISPLAY_NAME));
16                contactsData.setName(name);
17
18                /**
19                 * 查找该联系人的phone信息
20                 * 在ContactsContract.CommonDataKinds.Phone.CONTENT_URI 中查询
21                 * 条件为ContactsContract.CommonDataKinds.Phone.CONTACT_ID = 上面查询到的ID
22                 */
23                Cursor phones = getContentResolver().query(ContactsContract.CommonDataKinds.Phone.CONTENT_URI, null,
24                        ContactsContract.CommonDataKinds.Phone.CONTACT_ID + "=" + contactId, null, null);
25                if (phones != null && phones.moveToNext()) {
26                    //获取该联系人的手机号码
27                    String number = phones.getString(phones.getColumnIndex(ContactsContract.CommonDataKinds.Phone.NUMBER));
28                    contactsData.setNumber(number);
29                    phones.close();
30                }
31
32                /**
33                 * 查找该联系人的email信息
34                 * 在ContactsContract.CommonDataKinds.Email.CONTENT_URI 中查询
35                 * 条件为ContactsContract.CommonDataKinds.Phone.CONTACT_ID = 上面查询到的ID
36                 */
37                Cursor emails = getContentResolver().query(ContactsContract.CommonDataKinds.Email.CONTENT_URI, null,
38                        ContactsContract.CommonDataKinds.Email.CONTACT_ID + "=" + contactId, null, null);
39                if (emails != null && emails.moveToNext()) {
40                    //获取该联系人的邮箱
41                    String email = emails.getString(emails.getColumnIndex(ContactsContract.CommonDataKinds.Email.DATA));
42                    contactsData.setEmail(email);
43                    emails.close();
44                }
45                contactsDatas.add(contactsData);
46            }
47            cursor.close();
48        }
49        Log.d("测试","开始打印");
50        for (ContactsData c : contactsDatas) {
51            Log.e("联系人", c.toString());
52        }
